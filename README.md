# Final-Project
Data Analyst Final Project - Nicko Farhan

# Dataset memiliki 100,000 baris dan 20 kolom
Informasi kolom:
1. Order ID = Kode spesifik per pesanan
2. Order Date = Tanggal pemesanan
3. Customer ID = Kode identifikasi unik per pelanggan
4. Customer Name = Nama pelanggan
5. Product ID = Kode identifikasi unik per produk
6. Product Name = Nama produk
7. Category = Kategori produk
8. Brand = Merk produk
9. Quantity = jumlah item yang dipesan
10. Unit Price = Harga per item
11. Discount = Potongan harga
12. Tax = pajak per pesanan
13. Shipping Cost = Biaya pengiriman
14. Total Amount = Total pembayaran (Unit price x Quantity x Discount)
15. Payment Method = Metode pembayaran
16. Order Status = Status pemesanan / pengiriman
17. State = Provinxi
18. City = Kota
19. Country = Negara
20. Seller ID = Kode identifikasi unik per penjual
